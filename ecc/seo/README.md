# SEO 分类说明

该目录提供搜索引擎优化、内容增长、市场研究与社交流量协同相关能力，面向技术 SEO 审计、页面优化、结构化数据、关键词映射、竞争情报与增长内容分发场景。

## 当前目录功能

- **技术 SEO 审计**：支持 crawlability、indexability、canonical、redirect、sitemap、robots 与 Core Web Vitals 等问题排查。
- **页面与内容优化**：支持 title、meta description、heading、结构化数据、关键词映射和内部链接优化。
- **市场与竞争研究**：支持市场规模、竞品对比、投资人/基金研究、技术趋势扫描与决策型摘要。
- **内容增长协同**：支持品牌语气、跨平台内容改写、社交图谱排序、线索发现与 X/Twitter API 集成。

## Agents 简要说明

- **seo-specialist**：用于技术 SEO 审计、页面优化、结构化数据、Core Web Vitals、sitemap / robots 问题和 SEO 修复计划。

## Skills 简要说明

- **brand-voice**：从真实内容中提炼稳定文风，供内容、外联和社交场景复用。
- **connections-optimizer**：整理和优化 X、LinkedIn 社交网络，辅助删减、补充与暖启动外联。
- **content-engine**：为 X、LinkedIn、TikTok、YouTube、newsletter 等平台生成原生内容系统和改写方案。
- **crosspost**：把同一主题内容适配后分发到 X、LinkedIn、Threads、Bluesky 等多个社交平台。
- **frontend-patterns**：提供 React、Next.js、状态管理、性能优化、表单与可访问性相关前端模式。
- **lead-intelligence**：用于高价值线索的发现、评分、暖路径挖掘与跨渠道触达。
- **market-research**：用于市场研究、竞品分析、投资人尽调、技术趋势扫描和决策型研究输出。
- **seo**：用于技术 SEO、页面优化、结构化数据、Core Web Vitals、关键词映射与内容策略。
- **social-graph-ranker**：对社交关系图进行排序和桥接分析，寻找高价值引荐路径。
- **x-api**：提供 X/Twitter API 的发帖、读流、搜索和分析集成能力。
