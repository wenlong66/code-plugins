# Social 分类说明

该目录提供社交平台运营、分发、关系网络与触达相关能力，面向内容传播、社交图谱维护、线索挖掘与消息协同场景。

## 当前目录功能

- **社交内容分发**：支持跨平台内容编排与发布相关工作流。
- **社交关系优化**：支持社交网络整理、排序与高价值连接发现。
- **线索与消息触达**：支持围绕 X、LinkedIn 等渠道的线索挖掘与消息协同。

## Skills 简要说明

- **brand-voice**：从真实内容中提炼稳定文风，供内容、外联和社交场景复用。
- **connections-optimizer**：整理和优化 X、LinkedIn 社交网络，辅助删减、补充与暖启动外联。
- **content-engine**：为 X、LinkedIn、TikTok、YouTube 等平台生成原生内容系统和改写方案。
- **crosspost**：把同一主题内容适配后分发到多个社交平台，而不是机械同发。
- **email-ops**：用于邮箱整理、草拟、发送校验与已发送邮件核验。
- **lead-intelligence**：用于高价值线索的发现、评分、暖路径挖掘与跨渠道触达。
- **messages-ops**：用于查看短信和私信、找验证码、检查会话并确认消息来源。
- **social-graph-ranker**：对社交关系图进行排序和桥接分析，寻找高价值引荐路径。
- **x-api**：提供 X/Twitter API 的发帖、读流、搜索和分析集成能力。
